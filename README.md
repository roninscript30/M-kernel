# M-Kernel

> A Unix-like operating system built from scratch to learn computer architecture, operating systems, and kernel engineering.

## About

M-Kernel is a personal learning project focused on understanding how a computer works from the moment it powers on to running a complete operating system.

The goal is not to replace Linux, but to rebuild its core concepts step by step and gain practical experience in low-level systems programming.



## Project Goals

* Learn x86 Computer Architecture
* Learn Operating System Internals
* Understand Memory Management
* Build a Unix-like Kernel
* Study Networking from the Kernel Level
* Contribute to the Linux Kernel

## Technologies

* x86 Assembly (NASM)
* C
* GNU Make
* QEMU
* GCC
* GDB

## Build

```bash
make build
```

## Run

```bash
make run
```
